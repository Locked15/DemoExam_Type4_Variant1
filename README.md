# DemoExam_Type4_Variant1 (ООО «Книжный Клуб»)

В рамках выполнения задания была разработана информационная система для ООО «Книжный клуб»:
* неавторизованный клиент и авторизованный клиент может просматривать товары и
формировать заказ;
* менеджер может просматривать товары, формировать и редактировать заказы;
* администратор может добавлять/редактировать/удалять товары, просматривать и
редактировать заказы.
 
## Инструкция

Для запуска приложения необходимо запустить файл "Application.exe".
В случае недоступности БД к проекту приложена соответствующая резервная копия.

## Использованные Инструменты

* Язык программирования — C# 11;
* СУБД — MySQL;
* Среда разработки — Visual Studio 2022 Preview.
